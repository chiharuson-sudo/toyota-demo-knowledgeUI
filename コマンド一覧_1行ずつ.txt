========================================
  GitHub プッシュ用コマンド（1行ずつコピペ）
========================================

※ ターミナルで「1行コピー → 貼り付け → Enter」を繰り返してください。
※ 7行目だけ、自分のリポジトリURLに書き換えてからコピーしてください。

----------------------------------------
1行目（フォルダに移動）
----------------------------------------

cd "c:\internship\豊田自動織機\ナレッジグラフ用"

----------------------------------------
2行目（Git 初期化）
----------------------------------------

git init

----------------------------------------
3行目（ファイルを追加）
----------------------------------------

git add .

----------------------------------------
4行目（初回コミット）
----------------------------------------

git commit -m "TICO ナレッジグラフ PoC 初回"

----------------------------------------
5行目（リモート登録）※URLを自分のリポジトリに書き換えてからコピー
----------------------------------------

git remote add origin https://github.com/chiharuson-sudo/toyota-demo-knowledgeUI.git

----------------------------------------
6行目（ブランチ名を main に）
----------------------------------------

git branch -M main

----------------------------------------
7行目（プッシュ）
----------------------------------------

git push -u origin main

========================================
以上です。1行ずつ順番に実行してください。
========================================
