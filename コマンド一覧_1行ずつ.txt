========================================
  GitHub プッシュ用コマンド（1行ずつコピペ）
========================================

※ ターミナルで「1行コピー → 貼り付け → Enter」を繰り返してください。
※ 4・5行目は自分の名前とメールに書き換え、7行目は自分のリポジトリURLに書き換えてからコピーしてください。

----------------------------------------
1行目（フォルダに移動）
----------------------------------------

cd "c:\internship\豊田自動織機\ナレッジグラフ用"

----------------------------------------
2行目（Git 初期化）
----------------------------------------

git init

----------------------------------------
3行目（ファイルを追加）
----------------------------------------

git add .

----------------------------------------
4行目（Git の名前を設定）※ Your Name を自分の名前に書き換え
----------------------------------------

git config --global user.name "Your Name"

----------------------------------------
5行目（Git のメールを設定）※ you@example.com を自分のメールに書き換え
----------------------------------------

git config --global user.email "you@example.com"

----------------------------------------
6行目（初回コミット）
----------------------------------------

git commit -m "TICO ナレッジグラフ PoC 初回"

----------------------------------------
7行目（リモート登録）※URLを自分のリポジトリに書き換えてからコピー
----------------------------------------

git remote add origin https://github.com/chiharuson-sudo/toyota-demo-knowledgeUI.git

----------------------------------------
8行目（ブランチ名を main に）
----------------------------------------

git branch -M main

----------------------------------------
9行目（プッシュ）
----------------------------------------

git push -u origin main

========================================
以上です。1行ずつ順番に実行してください。

■ 今回だけ：いまコミットで止まっている場合
  すでに 1〜3 行目を実行済みなら、次の2行だけ実行してから「6行目」の commit を実行してください。

  git config --global user.name "あなたの名前"
  git config --global user.email "あなたのメール@例.com"

  （GitHub のメールアドレスを使うと、コミットが GitHub のプロフィールと紐づきます）
